## ⚡ Features

* **Neo-Brutalist Design:** Bold typography, hard shadows (`shadow-brutal`), striking contrasting colors (vibrant primary red with dark borders), and explicit structural grids.
* **Mechanical System Boot Loader:** A custom "System Boot" loading screen that simulates a terminal initialization sequence with staccato progress steps and live technical logging.
* **Responsive Layout:** Completely optimized for mobile, tablet, and desktop views without breaking the harsh geometric aesthetic.
* **Serverless Contact Form:** Fully integrated with `FormSubmit` to securely dispatch emails straight to Gmail without requiring a dedicated backend server.
* **Dynamic Animations:** Uses `framer-motion` for reveal-on-scroll animations that perfectly complement the stepped "staccato" style of the UI.

## 🛠️ Tech Stack

* **Frontend Framework:** React 18 (via Vite for blazing-fast HMR and building)
* **Styling:** Tailwind CSS (extended with custom theme variables for Neo-Brutalism)
* **Icons:** `react-icons` (Simple Icons, Font Awesome)
* **Fonts:** Space Grotesk (Display), Be Vietnam Pro (Sans/Body), Space Mono (Terminal/Code)

## 🎨 The "Teakcn" Neo-Brutalism Aesthetic

The portfolio breaks away from standard soft gradients and glassmorphism by embracing raw structure:
- **No Blurs:** Elements rely on heavy `4px` and `8px` black borders.
- **Hard Drop Shadows:** Instead of soft diffusion, shadows are solid blocks (e.g., `4px 4px 0px #000`) emphasizing depth through physical layering.
- **High Friction:** Transitions and animations (like the boot loader) are intentionally mechanical and stepped.

## 🚀 Getting Started Locally

To run this project on your local machine:

### 1. Clone the repository
```bash
git clone https://github.com/jangra22/portfolio.git
cd portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the development server
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the application.
